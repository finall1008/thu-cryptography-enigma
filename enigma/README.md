# Enigma 机破解

## 特点及限制

- 采用图灵的方法
- 只支持给定转子内部接线、反射器接线、Ring Setting 的情况下破解出转子排序、转子初始位置、插线板设置
- 只支持给定三个转子，而不是全部 5 个
- 只支持 6 对插线板交换，更多或更少都不行
- 不保证实现完全正确（只能说应该做对了作业？），仅供参考

## 使用方法

仿照现有例子在 [`config.txt`](config.txt) 中设置好转子、反射器内部接线、Ring Setting 和明密文，运行 [`enigma.py`](enigma.py) 即可。
